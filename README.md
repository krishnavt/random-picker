# Random Picker

A free, fair, and instant tool to pick random names or items from a list.

## Use Cases
- **Teachers**: Randomly call on students
- **Giveaways**: Pick random contest winners
- **Events**: Select volunteers or team assignments
- **Games**: Fair randomization

## Features
- ✅ No signup required
- ✅ No ads or annoying popups
- ✅ Two pick modes: Keep in list or Remove after picking
- ✅ Instant results with animation
- ✅ Works on all devices

## How to Use
1. Paste names/items (one per line)
2. Click "Update List"
3. Click "Pick Random"
4. Done!
